What is a Computer Programming Language?


Computer programming languages include basically any way of telling a computer what to do, and can be anything from basically human readable instructions to raw binary, otherwise known as 1s and 0s. Newer programming languages often look similar to English with words like “if”, “and”, “not”, etc. which are easier to understand.


Why Did I Choose This Topic?


I am interested in this topic because I have a lot of experience in programming, as well as I plan to study programming for a career. My driving question was, How have computer programming languages evolved over time since their inception to today?

Compiled vs. Interpreted


With all these new programming languages around, which of them is actually the fastest? Computers work by converting code to something that it can understand, which can either be done before or after the program is ran. This is important because it makes certain languages are not as capable as others at being fast because of the different ways they work. If a language is compiled then it is capable of being almost as fast as possible, but if it is interpreted then it is speed capped by the interpreter.


The Obvious Answer


According to the article Computer Programming Languages, by Donald Shirer, from the Salem Press Encyclopedia of Science, "Since assembly language offers the most control over the computer and its memory storage, it is used for tasks where the utmost in execution in speed or or data efficiency is needed. It takes considerable experience, patience, and self-discipline to write a large program in assembly language…"


Conclusions


Ultimately, any language that is compiled and doesn’t have any major speed handicaps, like C or Rust, can be practically considered the fastest practical programming language.

So, with all of these developments made to programming languages, what jobs have been created? Within the past 3 decades, many jobs have sprung up out of computer science, here are a few examples.


Web Programmers


One of the main types of programming done today is web development. According to the article 11 Types of Programming Jobs from Indeed, "Web developers design and create websites. They're responsible for both how the site looks and how it functions. They test and evaluate a site to make sure it meets quality standards before its release. After the site gets published, web developers monitor the site's continued performance and troubleshoot any technical problems that might arise. They ensure that their websites and programs meet current security standards."



Data Scientists


Along with web programming, the article also said that, "A data scientist uses technology to collect, organize, evaluate and interpret large or complex sets of data. Data scientists often develop complex algorithms designed to help their clients gain the most relevant insights into their clients or business operations. Data scientist duties can vary depending on their industry. For instance, while a data scientist at a government agency might primarily analyze data related to political opinions, a data scientist in social media may spend the majority of their time evaluating metrics like click-through or share rates."
